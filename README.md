# PSICAID Automation Tests

Repositorio de automatización de pruebas para la aplicación PSICAID.

## Tecnologías utilizadas

- Playwright
- TypeScript
- Page Object Model
- Data Driven Testing

## Escenarios automatizados

- Login con credenciales válidas
- Login con credenciales inválidas
- Validación de campos vacíos
